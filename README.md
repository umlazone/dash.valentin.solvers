# Solvers Mission Control

Minimal black/white ops board for the Solvers content factory (`@valentinflrz`).

## Stack
- Next.js 15 + Tailwind
- Deploy: Vercel
- Data v1: seeded UI (demo)
- Data v2: Supabase (`violetaAI`) — schema in `supabase/schema.sql`

## Local
```bash
npm install
npm run dev
```

## Aesthetic
Vercel/X-like: black background, white accents, monochrome badges, dense but readable ops UI.

## Status
First deployed demo target: read-only Mission Control with live account snapshot + draft queue.
